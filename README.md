# Corporate

A theme you can bring into a corporate environment without raising eyebrows.

Based on the [light variant of the excelent PaperColor neovim theme](https://github.com/NLKNguyen/papercolor-theme) by @NLKNguyen

<img width="2880" height="1800" alt="Theme Preview" src="https://github.com/defer/omarchy-corporate-theme/blob/main/preview.png" />

# Installation

`omarchy-theme-install https://github.com/defer/omarchy-corporate-theme.git`

Or, via walker:
1. `SUPER+ALT+SPACE`, Install > Style > Theme
2. Paste `https://github.com/defer/omarchy-corporate-theme.git`
3. Return
